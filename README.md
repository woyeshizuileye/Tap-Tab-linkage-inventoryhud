# Tap Tab

**这个模组能双击tab键<kbd>⇥ Tab</kbd> 来循环你的背包和物品栏，且支持自定义改键**  
**这个是我fork的，主要用作实现inventoryHud+这个模组的动画联动或者修复一些我觉得不好用的小bug。**  
当然，目前也仅仅只是实现了一点功能，还没优化（**又懒又菜又爱玩 哎**）
